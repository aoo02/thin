Listed below are the fonts listed in the variables:
    Hack Source- Required by the KDE Plasma Desktop Environment and it's the font I use for monospace stuff.
    Monaco - Former default monospace font for macOS. It's used for Mac compatibility.
    Consolas - Used for Windows compatibility. Default font when using Notepad on Windows 8+.
    Courier - Last fallback before "monospace". Only here because it's the second last used for Discord.
    Whitney - Discord's previous font for most stuff. It was replaced by "gg sans" in 2022 and quite a few people had a hissy fit about it. It can be re-enabled within the theme file.
